ANÁLISIS ELÁSTICO
Elastic Analysis — Comportamiento lineal σ = E·ε

El análisis elástico lineal es el método más utilizado en el diseño estructural. Asume que la relación entre tensión y deformación es lineal (Ley de Hooke) en toda la estructura. Las deformaciones son pequeñas, el material no plastifica y se aplica el principio de superposición. La gran mayoría de los programas de cálculo (ETABS, SAP2000, TEKLA) utilizan este método.

HIPÓTESIS FUNDAMENTALES
σ = E·ε [K]{u} = {F} (sistema lineal)
[K] = matriz de rigidez global de la estructura
{u} = vector de desplazamientos nodales
{F} = vector de fuerzas nodales externas
Hipótesis: material lineal · pequeñas deformaciones · superposición válida
Resultado: sobreestima resistencia en zonas plastificadas (conservador para diseño).
HIPÓTESIS
Lineal elástico
SUPERPOSICIÓN
Válida
SOFTWARE
ETABS · SAP2000 · TEKLA
USO
Diseño rutinario
  • 01Definir geometría, secciones y materiales.
  • 02Aplicar cargas y combinaciones según normativa.
  • 03Resolver [K]{u}={F} → obtener desplazamientos.
  • 04Calcular fuerzas internas N, V, M por compatibilidad.
  • 05Verificar ELU (resistencia) y ELS (servicio).

ANÁLISIS PLÁSTICO
Plastic Analysis — Mecanismo de colapso y redistribución

El análisis plástico considera la redistribución de momentos que ocurre cuando secciones altamente solicitadas plastifican (forman rótulas plásticas), permitiendo que otras secciones absorban mayor carga. La estructura colapsa cuando se forma un mecanismo con suficientes rótulas. Aprovecha la reserva plástica más allá del límite elástico.

CARGA DE COLAPSO PLÁSTICO
P_p = n · Mp / L_ef (n = número de rótulas)
Mp = fy · Zp = momento plástico de la sección [kN·m]
n = número de rótulas plásticas para el mecanismo
Teorema del límite superior: la carga de colapso real ≤ carga calculada por el mecanismo.
Teorema del límite inferior: equilibrio + Mp no superado → carga ≤ carga de colapso.
RÓTULAS
Donde M = Mp
RESERVA
f = Mp/Me ≥ 1.0
APLICA A
Acero / H.A. dúctil
NORMA
EC3 / AISC

ANÁLISIS DINÁMICO
Dynamic Analysis — Respuesta a cargas que varían con el tiempo

El análisis dinámico considera los efectos de inercia y amortiguamiento al evaluar la respuesta de la estructura ante cargas que varían con el tiempo. Es necesario cuando las frecuencias de excitación son cercanas a las frecuencias naturales de la estructura (resonancia) o cuando la aceleración es significativa.

ECUACIÓN DE MOVIMIENTO (VARIOS GDL)
[M]{ü} + [C]{u̇} + [K]{u} = {F(t)}
[M] = matriz de masas
[C] = matriz de amortiguamiento (ξ ≈ 2–5% en H.A.)
[K] = matriz de rigidez
{F(t)} = vector de fuerzas que varían con el tiempo
Métodos de solución: integración directa (Newmark) o superposición modal.
VARIABLES
u, u̇, ü (x, v, a)
AMORTIGUAMIENTO
ξ ≈ 2–5%
RIESGO
Resonancia (f = fn)
MÉTODO
Newmark / Modal

ANÁLISIS SÍSMICO
Seismic Analysis — Respuesta ante excitación del suelo

El análisis sísmico determina la respuesta estructural ante movimientos del suelo producidos por un terremoto. Puede realizarse por el método estático equivalente (corte basal), análisis espectral (superposición modal) o análisis tiempo-historia (integración directa con registros reales).

Espectro de diseño elástico reducido
MÉTODOS SÍSMICOS (NCh 433)
V = (S·A·I/R)·W | CQC de modos
Método estático equivalente: corte basal V = (S·A·I/R)·W — aplicable a estructuras regulares.
Análisis espectral: superposición de modos con espectro de diseño (CQC o SRSS).
Tiempo-historia: integración directa con acelerogramas reales o artificiales.
Vmin = 0.075·A·I·W/R (NCh 433 — cortante mínimo normativo)

ANÁLISIS MODAL
Modal Analysis — Modos y frecuencias naturales de vibración

El análisis modal determina las frecuencias naturales (fn) y modos de vibración (φ) de la estructura. Cada modo describe una forma deformada natural con su propia frecuencia. El análisis sísmico espectral combina la respuesta de cada modo mediante la regla CQC o SRSS para obtener la respuesta total.

PROBLEMA DE VALORES PROPIOS
([K] − ωₙ²·[M])·{φₙ} = {0}
ωₙ = frecuencia circular natural del modo n [rad/s]
fn = ωₙ/2π = frecuencia natural [Hz]
Tn = 1/fn = período natural [s]
{φₙ} = forma modal (vector propio) del modo n
Masa participante ≥ 90% → número mínimo de modos a considerar (NCh 433)
RESULTADO
T₁, T₂, … , Tn
MODOS
≥ 90% masa part.
COMBINACIÓN
CQC (correlados)
T₁ aprox. edif.
T₁ ≈ 0.1·N pisos

ANÁLISIS MATRICIAL
Matrix Structural Analysis — Base computacional del cálculo estructural

El análisis matricial de estructuras es el método computacional que formula el problema estructural como un sistema de ecuaciones lineales [K]{u}={F}. Es la base de todos los programas de análisis estructural. Permite analizar estructuras hiperestáticas complejas con cualquier geometría de manera sistemática.

SISTEMA GLOBAL DE ECUACIONES
[K]_{n×n} · {u}_{n×1} = {F}_{n×1}
[K] = matriz de rigidez global (ensamblada de matrices elementales)
{u} = vector de desplazamientos (GDL activos)
{F} = vector de cargas nodales externas
Solución: {u} = [K]⁻¹·{F} → fuerzas internas por compatibilidad
GDL = grados de libertad por nodo × número de nodos
BASE
[K]{u}={F}
ENSAMBLAJE
Matrices elementales → global
GDL barra 2D
6 (3 por nodo)
SOFTWARE
ETABS · SAP2000 · RFEM
  • 01Discretizar la estructura en elementos y nodos.
  • 02Calcular la matriz de rigidez de cada elemento [ke].
  • 03Ensamblar [K] global sumando las contribuciones.
  • 04Aplicar condiciones de borde (apoyos → eliminar filas/columnas).
  • 05Resolver {u}=[K]⁻¹{F} → calcular fuerzas internas.

MÉTODO DE DESPLAZAMIENTOS
Stiffness Method — Desplazamientos como incógnitas primarias

El método de desplazamientos (o método de la rigidez) toma los desplazamientos nodales como incógnitas principales. A partir de ellos calcula las fuerzas internas mediante la relación de rigidez. Es el método general implementado en todos los programas modernos de análisis estructural.

RIGIDEZ DE BARRA — VIGA EULERIANA
[ke] = EI/L³ · [ 12 6L −12 6L ; 6L 4L² −6L 2L² ; −12 −6L 12 −6L ; 6L 2L² −6L 4L² ]
La matriz [ke] relaciona fuerzas nodales y desplazamientos nodales del elemento.
DOF de viga 2D: translación y + rotación θ en cada extremo → 4 GDL
Rigidez axial se agrega aparte: EA/L · [1 −1; −1 1]
Compatibilidad: desplazamientos iguales en nodos compartidos.
INCÓGNITAS
Desplazamientos {u}
MATRIZ
Rigidez [K]
HIPERESTATISMO
No limita el método

MÉTODO DE FUERZAS
Flexibility Method — Fuerzas hiperestáticas como incógnitas

El método de fuerzas (o método de la compatibilidad) toma las fuerzas internas hiperestáticas como incógnitas. Se libera la estructura hasta hacerla isostática (estructura base) y se imponen condiciones de compatibilidad para encontrar las fuerzas redundantes. Es ideal para estructuras con pocas incógnitas hiperestáticas.

ECUACIÓN DE COMPATIBILIDAD
[f]·{X} = −{Δ₀} → {X} = −[f]⁻¹·{Δ₀}
[f] = matriz de flexibilidad (coeficientes de influencia)
{X} = vector de fuerzas hiperestáticas (incógnitas)
{Δ₀} = desplazamientos en la estructura isostática base bajo las cargas
Grado de hiperestatismo: h = b + r − 2n (número de incógnitas X)
Procedimiento clásico: integración de Mohr / Teorema de Castigliano.
INCÓGNITAS
Fuerzas redundantes {X}
MATRIZ
Flexibilidad [f]
VENTAJA
Pocas incógnitas
HERRAMIENTAS
Mohr / Castigliano
Diferencia clave con el Método de Desplazamientos: el método de fuerzas funciona mejor cuando el grado de hiperestatismo es bajo. El método de desplazamientos es más sistemático para cualquier nivel de complejidad y es el que usa el software actual.
