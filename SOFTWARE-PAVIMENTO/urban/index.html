Acera y Paso de Peatones
Escena Urbana 3D — Entorno Arquitectónico
Modelo 3D de alta fidelidad de una escena urbana contemporánea que incluye aceras, paso de peatones (cruce de cebra), pavimento asfaltado, vegetación y elementos peatonales. El modelo presenta múltiples materiales PBR con texturas reales de alta resolución para pavimento, concreto, terrazo y superficies pulidas, logrando una representación realista del entorno urbano.
