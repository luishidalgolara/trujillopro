◆ Los 3 diagramas esenciales en cálculo estructural ◆

Estos tres diagramas son la base de todo análisis estructural. Dominarlos es obligatorio.

#1
ESENCIAL
DMF
Diagrama de
Momento Flector
Muestra la distribución del momento interno a lo largo de vigas y columnas. Fundamental para el dimensionamiento de secciones de hormigón armado y acero.
Ecuación base M(x) = R·x − q·x²/2
Momento Vigas Marcos Flexión
#2
ESENCIAL
DFC
Diagrama de
Fuerza Cortante
Representa las fuerzas de corte transversal en cada sección del elemento. Determina la necesidad de estribos y refuerzo transversal en vigas de H.A.
Relación fundamental V(x) = dM/dx
Cortante Estribos Corte Transversal
#3
ESENCIAL
DFA
Diagrama de
Fuerza Axial
Muestra las fuerzas normales de compresión y tracción a lo largo del elemento. Esencial para el diseño de columnas, pilares y elementos de celosía.
Tensión normal σ = N / A
Axial Columnas Compresión Tracción
◆ Diagramas complementarios ◆

Herramientas adicionales para análisis estructural completo.

DC
Diagramas de Carga
Carga puntual, distribuida uniforme y triangular. Base para calcular reacciones y diagramas de esfuerzo.
Cargas Puntual Distribuida
DD
Diagrama de Deformación
Flecha máxima y línea elástica. Verifica estados límite de servicio (ELS) según NCh 430/ACI 318.
Flecha ELS Elástica
DI
Diagrama de Interacción M-N
Curva de capacidad para columnas de H.A. Verifica si la combinación M+N está dentro del dominio resistente.
M-N Columnas H.A.
DCL
Diagrama de Cuerpo Libre
Aísla elementos estructurales para análisis de equilibrio. Identifica todas las fuerzas y momentos actuantes.
Equilibrio Reacciones Aislamiento
CC
Camino de Cargas
Visualiza el flujo de cargas desde la losa hasta la fundación: Losa → Viga → Pilar/Muro → Fundación. Comprende cómo se distribuyen las cargas gravitacionales y sísmicas en toda la estructura.
Flujo Losa Viga Pilar Fundación
